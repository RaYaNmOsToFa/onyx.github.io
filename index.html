MapReduce Framework

Python • Concurrency • Networking

A lightweight MapReduce runtime that spawns worker processes or threads to execute map and reduce tasks, coordinates shuffle via TCP/UDP, and stores intermediate results to disk. The implementation demonstrates practical considerations for fault tolerance, task retries, and worker coordination.

  • Provided examples: wordcount, inverted index, URL frequency.
  • Used OS concurrency primitives and plain sockets for inter-worker communication.
  • Includes build/run instructions and test harness scripts.
